StudyPulse App 截图存放说明
==============================

将以下截图按指定的文件名保存到此目录下，网页会自动加载显示。
如果没有对应截图，页面会显示占位框。

截图列表（共 12 张）：

widget.png                      主屏小组件截图                    任意（建议正方形）

获取截图的方法：
1. 在 Xcode 中运行 App（模拟器或真机）
2. 导航到目标界面
3. 使用模拟器菜单 File > Save Screenshot 或设备截图
4. 将截图重命名为上述文件名，放入此目录

注意：所有截图文件名区分大小写，请保持与列表中完全一致。
