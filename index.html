Core Features

一切学习数据，尽在掌握

StudyPulse 将成绩、错题、考试与健康数据统一管理，让你对自己的学习状态始终保持清晰的全局视角。

成绩模块

成绩追踪与可视化

支持多科目成绩录入，可自定义满分、原始分、排名（1–5）和重要程度。每次录入均可附加试卷照片作为凭证。

  • 交互式趋势图 — 每科平均分、最高/最低分一目了然
  • "需要关注的科目"智能提醒 — 均分 < 70% 或近期下滑 > 15 分
  • 得分率色彩映射 — ≥90% 绿 / ≥75% 蓝 / ≥60% 橙 / <60% 红
  • 支持按科目、时间段筛选和排序
◆ images/grade-tracking.png
错题模块

四区结构错题本

独创四区块结构（题目 / 错误原因 / 错误解法 / 正确解法），每个区块可独立添加照片、OCR 识别与 Markdown 预览。

  • Vision OCR 文字识别 — 支持中文和英文，一键提取
  • Markdown 实时预览 — 每文本区独立编辑/预览切换
  • 关联科目与来源 — 按考试、练习册分类检索
  • 智能复习建议 — 基于错题时间和科目优先级推荐
▦ images/mistake-notebook.png
考试模块

考试日程与智能提醒

支持单科考试与多科综合考试（如模拟高考）。关联系统日历，自动调度本地通知。

  • 考前多级提醒 — 30天 / 10天 / 5天 / 3天 / 1天
  • 关联错题 — 考前自动聚合同科错题，精准复习
  • 日历集成 — 一键添加至系统日历（EventKit）
  • "未录入成绩提醒"卡片 — 考后 3–7 天自动提示
◬ images/exam-schedule.png
健康模块

HRV 健康准备度

读取 Apple Watch 夜间 HRV（SDNN）数据，使用 14 天个性化基线与 Z-score 评估当日学习状态。

  • 三级展示 — 简洁建议 / 数据详情 / 趋势图表
  • Z-score 分类 — 优秀(z>1.0) / 正常 / 偏低(z<-1.0)
  • 隐私优先 — 数据仅存本地，不外传
  • 引导式授权 — 首次使用前完成 HealthKit 授权引导

Platform

iPhone 与 iPad 双端原生适配

StudyPulse 不是简单的 iPhone 放大版。针对 iPad 做了专项优化，充分利用大屏空间。



iPhone

  • 5 标签 TabView 底部导航
  • 全宽单列布局，适合单手操作
  • Sheet 弹窗录入成绩/错题/考试
  • 硬件键盘 Tab 切换 / 数字跳转


iPad

  • NavigationSplitView 侧栏导航
  • 两栏 LazyVGrid 首页布局（16pt 间距）
  • 居中约束内容（640–1100pt 最大宽度）
  • 自适应 HStack / GridColumns / CardPadding
  • 硬件键盘导航 + 中强度触觉反馈
Getting Started

快速上手

从下载到开始记录第一条成绩，只需四步。

1

选择教育体系

在"设置 → 学术信息"中选择你的教育阶段与地区，科目列表将自动填充。

2

完善个人资料

填写姓名、学校、年级、目标院校等信息，让 App 更懂你的学习背景。

3

录入首条成绩

在首页点击"+"或从快速操作卡片进入，选择科目、填写分数、可附上试卷照片。

4

自定义主页

在"设置 → 主页布局"中拖拽排序、开关卡片，打造属于你的专属 Dashboard。

StudyPulseApp (@main)
|
├── LagMonitor ─────────────────────────────> LogStore
├── NotificationCoordinator (UNUserNotificationCenter delegate)
|
├── DataManager.asyncInit()
|   └── DataFileIO (nonisolated enum, ~/Documents/)
|       ├── profile / grades / subjects / mistakes / exams
|       ├── comprehensiveExams / health_history / srs_decks
|       └── images/ (grade/mistake/avatar jpg)
|
├── HealthKitManager.bootstrap()
|   ├── HKHealthStore (HRV SDNN, 心率, 呼吸率, 睡眠, 锻炼)
|   ├── HealthHistoryStore → health_history.json (30天滚动, NSLock)
|   └── StudyReadinessAlgorithm

└── scenePhase.active: sync all widgets
