核心功能 Core Features

一切学习数据，尽在掌握

StudyPulse 集成了 10+ 功能模块，覆盖成绩、错题、考试、计时、记忆、成就与健康评估。以下是核心功能概览。

◆

成绩追踪与可视化 Grade Tracking

分分计较。 Every mark matters.

多科目录入，自定义满分与排名。交互式趋势图 + "需要关注科目"智能提醒 + 得分率色彩映射。

▦

错题本 & OCR Mistake Notebook

错有错着。 Mistakes made, lessons saved.

四区结构（题目/错误原因/错误解法/正确解法），Vision OCR + 流式 Markdown 实时预览。

◬

考试管理与提醒 Exam Planner

从容赴考。 Walk in ready.

单科与综合考试，关联系统日历。考前 5 级通知，自动关联错题。支持时间段与跨日考试。

⌚

学习计时器 Study Timer

时尽其用。 Every second counts.

专注计时 + 灵动岛 Live Activity，横屏沉浸模式，空闲检测，与 HRV 健康数据联动。

▤

间隔重复记忆卡 SRS Flashcards

念念不忘。 What you repeat, you keep.

SM-2 算法闪卡系统，翻转动画 + 四档评分。可视化复习日历，到期自动提醒。

★

成就 & 每日目标 Achievements & Goals

功不唐捐。 No effort is wasted.

里程碑徽章 + 每日指标 + Siri 快捷指令。学习数据自动转化为激励，支持语音交互。

成绩模块

成绩追踪与可视化

分分计较。 Every mark matters.

支持多科目成绩录入，可自定义满分、原始分、排名（1–5）和重要程度。每次录入均可附加试卷照片作为凭证。

  • 交互式趋势图 — 每科平均分、最高/最低分一目了然
  • "需要关注的科目"智能提醒 — 均分 < 70% 或近期下滑 > 15 分
  • 得分率色彩映射 — ≥90% 绿 / ≥75% 蓝 / ≥60% 橙 / <60% 红
  • 支持按科目、时间段筛选和排序

错题模块

四区结构错题本

错有错着。 Mistakes made, lessons saved.

独创四区块结构（题目 / 错误原因 / 错误解法 / 正确解法），每个区块可独立添加照片、OCR 识别与流式 Markdown 实时预览。

  • Vision OCR 文字识别 — 支持中文和英文，一键提取
  • SwiftStreamingMarkdown 实时渲染 — 打字即所得
  • 关联科目与来源 — 按考试、练习册分类检索
  • 智能复习建议 — 基于错题时间和科目优先级推荐

计时模块

学习计时器

时尽其用。 Every second counts.

专注计时 + 灵动岛 Live Activity + 横屏沉浸模式。支持自动空闲检测，学习结束自动关联 HRV 准备度数据。

  • Live Activity — 锁屏 / 灵动岛实时显示计时进度
  • 横屏沉浸 — 全屏计时 + 自定义颜色主题
  • 空闲检测 — 长时间无操作自动暂停，防止虚报
  • HRV 联动 — 学习结束自动记录身体状态

成就系统 & 每日目标

平台 Platform

iPhone 与 iPad 双端原生适配

StudyPulse 不是简单的 iPhone 放大版。针对 iPad 做了专项优化，充分利用大屏空间。



iPhone · 单手操作

  • 5 标签 TabView 底部导航
  • 全宽单列布局，适合单手操作
  • Sheet 弹窗录入成绩/错题/考试
  • 硬件键盘 Tab 切换 / 数字跳转


iPad · 大屏沉浸

  • NavigationSplitView 侧栏导航
  • 两栏 LazyVGrid 首页布局（16pt 间距）
  • 居中约束内容（640–1100pt 最大宽度）
  • 自适应 HStack / GridColumns / CardPadding
  • 硬件键盘导航 + 中强度触觉反馈
快速上手 Getting Started

快速上手

从下载到开始记录第一条成绩，只需四步。

1

选择教育体系 Pick Your System

在"设置 → 学术信息"中选择你的教育阶段与地区，科目列表将自动填充。Choose your education stage and region — subjects auto-populate.

2

完善个人资料 Set Up Profile

填写姓名、学校、年级、目标院校等信息，让 App 更懂你的学习背景。Tell the app who you are and where you're headed.

3

录入首条成绩 Log Your First Grade

在首页点击"+"或从快速操作卡片进入，选择科目、填写分数、可附上试卷照片。Tap "+" on Home, pick a subject, enter your score.

4

探索更多 Explore Further

开启学习计时、创建记忆卡片、解锁成就徽章、设置每日目标——打造你的完整学习工作流。Timer, flashcards, achievements — build your complete workflow.

架构概览

StudyPulseApp (@main)
|
├── LagMonitor ─────────────────────────────> LogStore
├── NotificationCoordinator (UNUserNotificationCenter delegate)
|
├── DataManager.asyncInit()
|   └── DataFileIO (nonisolated enum, ~/Documents/)
|       ├── profile / grades / subjects / mistakes / exams
|       ├── comprehensiveExams / health_history / srs_decks
|       └── images/ (grade/mistake/avatar jpg)
|
├── HealthKitManager.bootstrap()
|   ├── HKHealthStore (HRV SDNN, 心率, 呼吸率, 睡眠, 锻炼)
|   ├── HealthHistoryStore → health_history.json (30天滚动, NSLock)
|   └── StudyReadinessAlgorithm
|       ├── PersonalBaselines (30天个人基线, ≥7天启用)
|       └── AgeReference (年龄段参考, 样本不足时回退)
|
└── scenePhase.active: sync all widgets

Models ── Codable + Sendable value types
├── Subject / Grade / MistakeNote / Exam / comprehensiveExam
├── UserProfile / AppPreferences / HomeLayoutPreference
├── DailyHealthSnapshot / Flashcard / FlashcardDeck
└── StudyReport / SubjectSummary / HealthCorrelation...

Managers ── @MainActor ObservableObject / nonisolated
├── DataManager ── 中央枢纽, @EnvironmentObject
├── AppEnvironmentManager ── 语言 / 主题
├── HealthKitManager ── HRV / 身体状态
├── CalendarManager / OCRManager / ImageCache
├── DataExportManager (CSV) / ReportRenderer (PDF)
├── StudyTimerManager ── Live Activity + 空闲检测
├── SRSManager ── SM-2 间隔重复
├── AchievementManager / DailyGoalManager
├── WidgetDataSync / TrendWidgetSync / HRVWidgetSync
└── StudyReadinessAlgorithm ── 5档强度 × 5类重点

Views ── SwiftUI
├── [iPhone] TabView: Home / Trends / Mistakes / Exams / Settings
├── [iPad] NavigationSplitView + 侧栏
├── HomeView ── 8 可定制卡片 (HRV / 提醒 / 快捷 / 建议 / 图表 / 考试 / 名言 / 成绩)
├── FlashcardStudyView ── SM-2 间隔重复
├── Intents ── Siri Shortcuts (AddGrade / CheckReadiness / ...)
└── Widgets: ExamWidget / TrendWidget / HRVWidget

Persistence
├── ~/Documents/ ── 8 个 JSON + images/ 目录
