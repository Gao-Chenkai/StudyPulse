成绩模块

成绩追踪与可视化

分分计较。 Every mark matters.

支持多科目成绩录入，可自定义满分、原始分、排名（1–5）和重要程度。每次录入均可附加试卷照片作为凭证。

  • 交互式趋势图 — 每科平均分、最高/最低分一目了然
  • "需要关注的科目"智能提醒 — 均分 < 70% 或近期下滑 > 15 分
  • 得分率色彩映射 — ≥90% 绿 / ≥75% 蓝 / ≥60% 橙 / <60% 红
  • 支持按科目、时间段筛选和排序

错题模块

四区结构错题本

错有错着。 Mistakes made, lessons saved.

独创四区块结构（题目 / 错误原因 / 错误解法 / 正确解法），每个区块可独立添加照片、OCR 识别与流式 Markdown 实时预览。

  • Vision OCR 文字识别 — 支持中文和英文，一键提取
  • 流式 Markdown 渲染 — SwiftStreamingMarkdown 实时预览
  • 关联科目与来源 — 按考试、练习册分类检索
  • 智能复习建议 — 基于错题时间和科目优先级推荐

考试模块

考试日程与智能提醒

从容赴考。 Walk in ready.

支持单科考试与多科综合考试（如模拟高考），可设置时间段与跨日考试。关联系统日历，自动调度本地通知。

  • 考前多级提醒 — 30天 / 10天 / 5天 / 3天 / 1天
  • 关联错题 — 考前自动聚合同科错题，精准复习
  • 日历集成 — 一键添加至系统日历（EventKit）
  • "未录入成绩提醒"卡片 — 考后 3–7 天自动提示

健康模块

身体状态 & 学习准备度

身有数，学有度。 Your body knows. Learn accordingly.

基于 Apple Watch 健康数据：HRV（SDNN）、静息心率、呼吸率、睡眠分期、锻炼时长。通过个性化 30 天基线与 Z-score 评估当日学习状态。

  • 5 维评估 — HRV / 心率 / 呼吸 / 睡眠 / 运动
  • 25 种组合建议 — 5 档强度 × 5 类重点，精准匹配
  • 三级展示 — 简洁建议 / 数据详情 / 趋势图表
  • 隐私优先 — HealthKit 数据仅存本地，不上传

计时模块

学习计时器

时尽其用。 Every second counts.

专注计时 + 灵动岛 Live Activity + 横屏沉浸模式。支持自动空闲检测，学习结束自动关联 HRV 准备度数据。

  • Live Activity — 锁屏 / 灵动岛实时显示计时进度
  • 横屏沉浸 — 全屏计时 + 自定义颜色主题
  • 空闲检测 — 长时间无操作自动暂停，防止虚报
  • HRV 联动 — 学习结束自动记录身体状态

记忆模块

间隔重复记忆卡

念念不忘。 What you repeat, you keep.

SM-2 算法驱动的闪卡系统，自动调度复习间隔。多卡组管理 + 翻转动画 + 复习结果追踪。

  • SM-2 算法 — 根据评分自动调整间隔 (ease factor)
  • 卡组管理 — 按科目创建卡组，正面/背面双面设计
  • 翻转动画 — 点击翻转 + 四档评分 (重来 / 困难 / 良好 / 轻松)
  • 复习日历 — 可视化复习计划，到期卡片自动提醒

激励模块

成就系统 & 每日目标

功不唐捐。 No effort is wasted.

里程碑成就 + 每日学习目标 + Siri 快捷指令。学习数据自动转化为可视化激励，支持语音交互。

  • 成就系统 — 录入成绩、复习卡片、专注计时解锁徽章
  • 每日目标 — 自定义每日学习指标，进度实时展示
  • Siri 快捷指令 — 语音查分 / 查考试 / 记错题 / 问状态
  • PDF 学习报告 — 一键生成含图表的综合学习报告
